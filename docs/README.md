# 문서 디렉토리

프로젝트의 모든 문서를 통합 관리하는 디렉토리입니다.

## 📁 구조

```
docs/
├── analysis/          # 분석 결과 및 전처리 문서
│   ├── hs4_analysis_report.md
│   ├── hs4_item_analysis_summary.md
│   ├── preprocessing_improvements.md
│   └── README_ANALYSIS.md
├── models/            # 모델 기술 문서
│   ├── MODEL_COMPARISON.md
│   ├── MODEL_PREPROCESSING.md
│   ├── MODEL_EV_BATTERY.md
│   ├── IMPROVEMENTS_*.md
│   ├── PERFORMANCE_REPORT.md
│   ├── EXECUTION_SUMMARY.md
│   └── README.md
└── project/           # 프로젝트 전체 문서
    ├── supply_chain_model_summary.md
    ├── tier_explanation.md
    ├── notion_prompt_final.md
    └── notion_organization_prompt.md
```

## 📚 문서 분류

### analysis/ - 분석 관련 문서
- **분석 결과**: HS4 코드 분석, Item 분석 등
- **전처리 문서**: 전처리 개선사항, 방법론

### models/ - 모델 기술 문서
- **모델 가이드**: 모델 비교, 사용법, 상세 설명
- **개선 문서**: 개선사항 체크리스트, 요약
- **성능 리포트**: 성능 비교, 실행 요약
- **기술 문서**: 평가 지표 분석, Cross-Validation 설명

### project/ - 프로젝트 전체 문서
- **프로젝트 요약**: 공급망 모델 요약, 계층 설명
- **외부 공유용**: Notion 정리용 프롬프트
