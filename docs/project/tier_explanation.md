# 공급망 계층 구조 (Tier) 설명

## 🔄 공급망 계층 구조

기계산업 공급망은 **원자재 → 중간재 → 최종제품**의 3단계 구조로 이루어져 있습니다.

```
┌─────────────────────────────────────────┐
│  Tier 1: 원자재 (Raw Materials)         │
│  화학제품 (HS4: 2800-3899)              │
└──────────────┬──────────────────────────┘
               │ 가공/제조
               ▼
┌─────────────────────────────────────────┐
│  Tier 2: 중간재 (Intermediate Goods)    │
│  금속제품 (HS4: 7200-8399)              │
└──────────────┬──────────────────────────┘
               │ 조립/제조
               ▼
┌─────────────────────────────────────────┐
│  Tier 3: 최종제품 (Final Products)      │
│  기계류, 운송기기, 정밀기기              │
│  (HS4: 8400-9299)                       │
└─────────────────────────────────────────┘
```

---

## 📦 Tier 1: 원자재 - 화학제품

### 정의
- **역할**: 기계산업의 가장 기본이 되는 원자재
- **HS4 범위**: 2800-3899 (화학공업 제품)
- **특징**: 다른 제품을 만드는 데 필요한 기본 화학 물질

### Tier 1의 특징
- **무역량 비중**: 67.43% (가장 큼)
- **무게 비중**: 87.36% (가장 큼)
- **역할**: 기계산업의 기초 원료 제공

### 예시
- 암모니아(2814) → 비료 제조 → 농업 기계 부품
- 니켈(2805) → 배터리 제조 → 전기차 부품
- 화학제품(3824) → 플라스틱/고무 제조 → 기계 부품

## 🔧 Tier 2: 중간재 - 금속제품

### 정의
- **역할**: 원자재를 가공하여 만든 중간 제품
- **HS4 범위**: 7200-8399 (금속 및 금속제품)
- **특징**: 최종제품을 만들기 위한 부품/소재

| **7202** | 철합금 | 0.41% | 합금강 |

### Tier 2의 특징
- **무역량 비중**: 3.37%
- **무게 비중**: 6.47%
- **역할**: 원자재를 가공하여 기계 부품 소재로 변환

### 예시
- 철강재(7207) → 기계 부품 제조 → 최종 기계 제품

## 🏭 Tier 3: 최종제품

- **HS4 범위**:

## 🔄 공급망 흐름 예시

### 예시 1: 전기차 배터리
```
Tier 1 (원자재)
  ↓ 니켈(2805) + 코발트(8105)
Tier 2 (중간재)
  ↓ 니켈(8102) + 코발트(8105) → 배터리 셀
Tier 3 (최종제품)
  ↓ 전자석(8505) + 전기모터(8501) → 전기차
```

### 예시 2: 공작기계
```
Tier 1 (원자재)
  ↓ 화학제품(3824) → 플라스틱/고무
Tier 2 (중간재)
  ↓ 철강재(7207) → 기계 부품
Tier 3 (최종제품)
  ↓ 공작기계(8461) → 제조업 장비
```

## 💡 핵심 포인트

1. **Tier 1 (원자재)**: 
   - 가장 큰 비중 (67.43%)
   - 화학제품이 기계산업의 기초
   - 예: 암모니아, 비료, 화학제품

2. **Tier 2 (중간재)**:
   - 원자재를 가공한 부품 소재
   - 예: 철강재, 니켈, 코발트

3. **Tier 3 (최종제품)**:
   - 완성된 기계 제품
   - 예: 전기모터, 전자석, 공작기계

4. **공급망 방향성**:
   - Tier 1 → Tier 2 → Tier 3 순서로 흐름
   - 하위 계층의 변화가 상위 계층에 영향
